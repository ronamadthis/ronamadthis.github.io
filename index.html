Projects


Created and compared six ML models on stroke data of 43400 rows. Models used were: Logistic Regression, Decision Tree Classifier, K Neighbors, SVC, Linear Discriminant Analysis and Gaussian NB. k-fold cross-validation with k=10 was used in these models. Accuracy scores were obtained and plotted. Best model was selected based on accuracy score. Key parameters that predict stroke were identified.  Code


About 1200 combinations of latitudes and longitudes were randomly generated. Citipy python library was used to identify at least 500 unique cities based on geographical coordinates. Then, OpenWeatherMap API was used to obtain weather parameters (Temperature, humidity, cloudiness and wind speeds) of these cities. The weather parameters were plotted against latitude to study any dependencies.  Code
